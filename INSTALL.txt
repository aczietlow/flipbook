To install this module you must:

1.  Download the module
2.  Extract the module into drupal's module folder
    [sites/all/modules or sites/all/modules/contrib for example]
3.  Enable the module
4.  If you have not yet extracted the plugin's files do so at:
    sites/all/libraries/real-3d-flipbook

    The resulting structure should look like:
    sites/all/libraries/real-3d-flipbook/deploy
    sites/all/libraries/real-3d-flipbook/documentation
    sites/all/libraries/real-3d-flipbook/source

    Javascript files should be at:
    sites/all/libraries/real-3d-flipbook/source/js
